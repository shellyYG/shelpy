## Technologies
### Back-End
- Node.js / Express.js
- RESTful API
- Linux
- NGINX
- Socket.IO

### Front-End
- React & Redux
- HTML
- CSS
- JavaScript
- Axios

### Cloud Service (AWS)
- Elastic Compute Cloud (EC2)
- Relational Database Service (RDS)

### Database
- MySQL

### Networking
- HTTP
- HTTPS / SSL
- Domain Name System (DNS)

### Others
- Sendgrid API (Triggered emails)
- Microsoft Teams API (Meeting links)
- TapPay API (Credit card payment)
- Design Pattern: MVC
- Version Control: Git, GitHub
- Agile: Trello (Srum)
- Test: Mocha / Chai
